Maine Town Quiz

About this project

FAQ
Why did you make this?
I've wanted to learn all the towns in Maine for a long time and couldn't find a good way to do it. So I built one with Claude Code.
What towns are included? Why are some missing?
The quiz includes all incorporated towns and cities. Unorganized territories, townships, plantations, patents, and grants were outside the original scope of the quiz. I've included all the Gores because I find the concept charming.
Where does the map data come from?
Town boundaries come from the Maine Office of GIS, which maintains authoritative boundary files for the state. I'm grateful for their work making this data publicly available.
Does my progress get saved?
Yes. All scores and game history are saved in your browser's local storage. Progress is stored per device and per browser. Clearing your browser data will reset it.
How do I unlock Challenge Mode?
Score 100% on any county in Normal Mode. Once unlocked, you'll be offered a choice of Normal or Challenge Mode each time you select that county.
Will you add other states?
No, this quiz is specifically about Maine. But the code is open source and available on GitHub under a CC BY-NC license, so you're welcome to adapt it for any state you love.
